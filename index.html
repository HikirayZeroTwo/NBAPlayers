О проекте

Что это такое?

Проект создан для хранения и управления статистикой игроков NBA. Введите данные — и они сохранятся навсегда. Удобный поиск, сортировка, редактирование записей доступны как на компьютере, так и на телефоне.

📋

11 полей на игрока

Имя, фамилия, возраст, рост, команда, матчи, минуты, очки, FG%, 3P%, FT%

💾

Постоянное хранение

Десктоп сохраняет в Excel-файл, мобильное — в JSON на устройстве

🔍

Поиск и сортировка

Мгновенный поиск по любому полю, сортировка по очкам, матчам, фамилии

✏️

Полное CRUD

Добавляй, редактируй и удаляй записи в пару кликов

🖥 Десктоп

NBA Players Desktop

Программный модуль с полноценным GUI. Вводи данные в удобную форму, ищи игроков, сортируй список — всё сохраняется в Excel-файл на твоём компьютере.

Python 3 tkinter openpyxl Windows / macOS / Linux
Скачать .py файл
📱 Android

NBA Players Mobile

Мобильное приложение для Android. Те же 11 полей ввода, список с карточками, поиск и сортировка. Данные сохраняются в JSON прямо на телефоне.

Kotlin Android Studio RecyclerView Android 7.0+
Скачать APK (.apk)
